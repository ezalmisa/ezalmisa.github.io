Project Summary

This is the concept design project of the web based application for users especially auditor to manage the audit inspection activities.

Project Goal

The main goal of this project is to come out with a solution help the auditors and other demographic groups to improve the efficiency of audit inspection activities to ensure their company product quality in the form of desktop version of web based application

In order to achieve the goal, we decided to incorporate features such as real time scheduling, digital inspection form that is accessible via online and offline, push notification system that could alert users to respond on the corrective actions produced from inspection activities, and report generation from the completed inspection that can be shared with all users within the organization.
